## Overview
This app displays a fun fact of the day that only shows once per day. The user will learn a new fact (preferably to start their day) that can be intriguing or not so interesting. The user can express their reaction to the fun fact by selecting a pre-determined response and then seeing the stats of how others reacted to it. I built this app as a beginner project to help learn computer programming and also to show creativity and originality.

## Features
Main Functionalities:
- New Fun Fact each day
- Optional answers to show reaction
- Poll of users responses
- Responsive layout (works on mobile and desktop)

## Tech Stack
- **Language:** Typescript
- **Frontend:** React, Vite, Tailwind CSS, Radix UI, React Hook Form
- **Backend:** Node.js, Express
- **Database:** Drizzle ORM
- **Tools:** drizzle-kit, npm, Git, Replit
